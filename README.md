# Raytracer-CPP

Incredibly trivial ray-tracer written in plain, understandable C++

## Multi-Threading

Thanks to multi-threading, the following results rendering the Cover Scene (MVP 1) 2000 * 1000 is archived:

* real 7m50.069s
* user 50m52.002s
* sys 0m3.129s
